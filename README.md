# 株式会社inside - 公式ウェブサイト

M&A・IPO支援・企業価値評価の専門家集団

## 📱 プロジェクト概要

株式会社insideの公式コーポレートサイトです。M&A、IPO支援、デューデリジェンス、企業価値評価を専門とする会計事務所として、確かな実績と高度な専門知識で企業の成長をサポートします。

### 🎯 サイトの特徴

- **モダンなデザイン**: プロフェッショナルで洗練されたビジュアル
- **完全レスポンシブ**: PC、タブレット、スマートフォンに完全対応
- **スムーズなアニメーション**: 視覚的に魅力的なユーザー体験
- **SEO最適化**: 検索エンジンに最適化されたHTML構造
